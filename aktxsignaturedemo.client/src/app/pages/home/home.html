Signature Demo

A sample application that shows how to integrate TX Text Control's document editor, viewer, and electronic signature components into an Angular frontend backed by ASP.NET Core and .NET Aspire. Browse the pages to see each component in action.

Open Editor Sign a Document

What's Inside

Each page demonstrates a different TX Text Control component

Document Editor

Embeds the TX Text Control WYSIWYG editor via WebSocket. You can create, open, and edit documents, place signature fields, and save back to the server.

Open Editor

Document Viewer

Uses the TX Text Control DocumentViewer middleware to display documents read-only with thumbnail navigation and text search.

Open Viewer

Electronic Signatures

Loads a document with signature fields into the viewer and lets users sign with typed, drawn, or uploaded signatures. The signed result is returned as PDF.

Sign Document

Tech Stack

Angular 19, ASP.NET Core on .NET 10, orchestrated with .NET Aspire. TX Text Control v34 handles all document processing on the server side.

.NET Aspire
